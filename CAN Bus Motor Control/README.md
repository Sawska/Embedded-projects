### **Project 3 — CAN Bus Motor Control**

**Description:**  
Use two ESP32 boards to communicate over CAN bus to control a motor.

**BOM:**

- 2× ESP32
    
- MCP2515 CAN module × 2
    
- L298N motor driver
    
- DC motor
    

**Scope Mapping:**

- **CAN Protocol:** Multi-device communication.
    
- **Motor Control:** Remote commands.
